## 📦 Bitácora & Cheat Sheets Operativas 🛠️

<details>
  <summary><b>💡 Haz clic aquí para ver mi documentación rápida de comandos</b></summary>
  <br>
  
  ### 🐧 Administración Linux
  * `sudo apt update && sudo apt upgrade -y` - Sincronización de repositorios y actualización del sistema.
  * `chmod +x script.sh` - Asignación de permisos de ejecución para scripts de automatización.
  
  ### 🌐 Diagnóstico de Redes (Sistemas)
  * `ipconfig /all` - Volcado completo de la configuración de red en entornos Windows.
  * `ping 8.8.8.8` - Diagnóstico básico de conectividad e ICMP hacia DNS externo.
