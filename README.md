## 🧭 Sobre mí
Soy técnica en SMR con vocación por el **desarrollo web aplicado a sistemas**. Mi trabajo diario se centra en diagnosticar hardware, configurar redes y crear herramientas web que facilitan la administración técnica.

## ⚙️ Mi Workflow y Herramientas
* **Versionado:** Git y GitHub (flujo de ramas controlado).
* **Diagnóstico:** Uso de Wireshark para redes y herramientas de testeo para hardware.
* **Entornos:** Virtualización con VirtualBox (laboratorios aislados).
* **Editor:** VS Code con extensiones de productividad configuradas.
* **Documentación:** Markdown puro para cada bitácora de aprendizaje.
